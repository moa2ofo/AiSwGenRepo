Detailed Description

ECU temperature calculation implementation.

This source file implements Calc_ecuTemperature_s16, which computes an ECU temperature estimate by combining an NTC-based measurement with a system temperature measurement according to a piecewise algorithm controlled by PAR_ECU_TEMP_LOW_THR and PAR_ECU_TEMP_UP_THR.

sint16 Calc_ecuTemperature_s16 ( void )

Compute ECU temperature estimate from NTC and system temperatures.

Goal of the function

The purpose of this function is to provide a robust estimate of the ECU temperature by combining:

  • An NTC-based temperature measurement (fast, but can fail or be noisy).
  • A system temperature measurement (more stable, backup in case of NTC fault).
